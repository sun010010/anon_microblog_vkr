# Анонимный микроблог

Одностраничное веб-приложение для анонимного общения с защитой от XSS, CSRF, SQL-инъекций и брутфорса.

## Стек технологий
- **Backend:** Python, Flask, SQLite
- **Frontend:** HTML, CSS, JavaScript
- **Безопасность:** JWT, bcrypt, httpOnly cookies, rate limiting, HTTPS

## Основные функции
- Регистрация и вход
- Публикация постов (до 500 символов)
- Комментарии (до 300 символов)
- Лайки и дизлайки
- Закладки
- Поиск по тексту постов
- Фильтр «Все посты» / «Мои посты»
- Светлая и тёмная тема
- Редактирование своих постов
